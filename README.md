About
This is a simple Java desktop application for project registration. It uses Java Swing for the interface and saves project data to a local text file.

How to Run
I used only the standard Java library. 
You can run it from the terminal:  
Compile: javac *.java
Run: java ProjectFormApp   


Steps in App
Input: Enter the project name, team leader, and start date.  
Select: Choose the team size and project type from the menus.  
Save: The app checks if fields are full and saves the data to projects.txt.  
Time: It automatically adds the current date and time to every record.  
Clear: Reset all fields to start a new entry.
